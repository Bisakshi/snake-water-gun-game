# 🐍 Snake Water Gun Game

## 📌 Introduction

This is a simple Python-based game inspired by the classic **Snake, Water, Gun** concept (similar to Rock-Paper-Scissors).
The user plays against the computer, and the winner is decided based on predefined rules.

---

## 🎮 Game Rules

* Snake drinks Water → Snake wins 🐍
* Water damages Gun → Water wins 💧
* Gun kills Snake → Gun wins 🔫

---

## 🚀 Features

* Interactive command-line gameplay
* Random choice generation using Python
* Simple and beginner-friendly logic
* Input validation for user choices

---

## 🛠️ Tech Stack

* Python 🐍
* Random module

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Bisakshi/snake-water-gun-game.git
   ```

2. Navigate to the project folder:

   ```bash
   cd snake-water-gun-game
   ```

3. Run the program:

   ```bash
   python game.py
   ```
